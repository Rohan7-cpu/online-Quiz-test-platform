# Quiz Platform (React + Redux + MUI)

This is a ready-to-run scaffold for the Quiz Platform assignment.

## Quick start

1. Extract the zip and open terminal in the project folder.
2. Run `npm install`
3. Run `npm start` to launch the dev server.
4. The app opens at http://localhost:3000

Features implemented:
- React + Redux Toolkit + Thunk
- React Router v6
- Material UI components
- Create / Edit / Delete quizzes stored in localStorage (key: `questions`)
- Play Quiz flow and Results
- Validations per assignment brief
- Responsive layout with MUI Grid
