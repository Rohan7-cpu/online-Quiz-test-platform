# Online Quiz Test Platform

A web-based quiz platform built with **React** and **Redux** that allows users to play quizzes, and admins to create and manage quizzes.

---

## **Features**

- User-friendly interface to play quizzes.
- Admin dashboard to create and manage quizzes.
- Track scores and results.
- Multiple-choice questions support.
- Role-based access (Admin / Player).
